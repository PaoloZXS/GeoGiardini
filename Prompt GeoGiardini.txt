📌 Titolo progetto
GeoGiardini – PWA Android + Backend TURSO + Hosting Vercel + Foto Dropbox + Notifiche + Gestione Lavori

🎯 OBIETTIVO GENERALE
Sviluppare una PWA installabile su Android, ospitata su Vercel, con database TURSO, che gestisca:

Giardinieri

Clienti

Lavori svolti

Messaggi e notifiche

Foto ridimensionate automaticamente e salvate su Dropbox

Ricerca avanzata dei lavori

Esportazione PDF

Versioning automatico

UI semplice, leggibile, tema verde/natura

L’app deve essere modulare, veloce, chiara, con file HTML/CSS/JS separati per ogni pagina.

🏗️ ARCHITETTURA TECNICA UFFICIALE
Frontend:

PWA con frontend statico

Hosting su Vercel

Backend:

API Routes serverless su Vercel

Nessun server dedicato

Tutta la logica backend deve essere implementata tramite funzioni serverless:

login

sessioni

query TURSO

upload Dropbox

notifiche push

generazione PDF

Il client non accede mai direttamente a TURSO o Dropbox.

Sicurezza, permessi e validazioni devono essere gestiti lato serverless.

Le credenziali tecniche e le chiavi di configurazione verranno fornite separatamente.

“Lavoro svolto dal giardiniere [NOME GIARDINIERE] per il cliente [NOME CLIENTE]: [DESCRIZIONE]. Note: [NOTE]. Data: [DATA ORA].”
